# Плакат AIRI 2026

Плакат представлен на русском и английском языках в эквивалентных портретных форматах A1:

- `Lapin_Vladislav_DoRA_AIRI_2026_RU.pptx` — редактируемый русский исходник PowerPoint;
- `Lapin_Vladislav_DoRA_AIRI_2026_RU.pdf` — русский экспорт ISO A1, готовый к печати (`594 × 841 mm`);
- `poster_preview_RU.png` — облегчённый предпросмотр русской версии;
- `Lapin_Vladislav_DoRA_AIRI_2026.pptx` — редактируемый английский исходник PowerPoint;
- `Lapin_Vladislav_DoRA_AIRI_2026.pdf` — английский экспорт ISO A1, готовый к печати;
- `poster_preview.png` — облегчённый предпросмотр английской версии.

Официальная шапка AIRI 2026, двухколоночная компоновка, логотипы партнёров и область QR-кода сохранены. Русская версия использует отдельные графики с суффиксом `_RU`, построенные из тех же провалидированных выходных данных проекта в `figures/extension/`:

1. `confirmatory_dora_minus_lora_RU` — парные оценки MLP/CNN на удержанных сидах (поправка Холма: `p=0.382213` и `0.158955`);
2. `data_regime_accuracy_RU` — перебор вложенных объёмов данных: 50/100/200/400 примеров;
3. `synthetic_optimization_RU` — допустимая представительная способность против фактической оптимизации.

QR-код явно обозначен как приватная ссылка для рецензирования и ведёт в этот репозиторий. Зрителям, которые не являются соавторами репозитория, потребуется запросить доступ; код проекта намеренно не меняет видимость репозитория.
